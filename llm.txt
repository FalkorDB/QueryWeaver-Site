# QueryWeaver

QueryWeaver is an open-source, graph-powered Text-to-SQL solution designed for enterprise databases. It transforms natural language questions into accurate SQL queries by understanding your database schema's complex relationships using FalkorDB.

## Core Features
- Graph-Powered Schema Understanding: Uses FalkorDB to model relationships for better joins and accuracy.
- Contextual Memory: Leverages Graphiti for chat history and agentic memory.
- Open Source: Built on FalkorDB and available on GitHub.
- Database Support: Works with MySQL and PostgreSQL.
